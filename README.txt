Cuadro Mágico

-----------------------------------------

Descripción:

Algoritmo genético configurable para resolver el problema del cuadro mágico.
La descripción teórica de este problema define que su solución es una matriz
donde todas sus filas, columnas y dos diagonales sumen lo mismo. Además los
valores de la matriz deben ser diferentes entre sí.

-----------------------------------------

Como funciona:

Cuando se muestra una posible solución, se pinta cada elemento de la matriz
de gris, si este no hace parte de una tupla entre las tuplas que suman
igual; azul si hace parte de una tupla entre las tuplas que suman igual; 
morado si hace parte de dos tuplas entre las tuplas que suman igual; y
fucsia si hace parte de tres tuplas entre las tuplas que suman igual.

La solución, según la directiva anterior, se pintaria con todos los
elementos morados, y las dos diagonales de fucsia.

La descripción de los resultados del algoritmo se mostrará cuando se pase
el puntero del mouse por encima del panel donde se muestra la posible
solución.

Se puede configurar los parámetros usados para el funcionamiento del
algoritmo en Configuraciones->Parámetros del Algoritmo. También se puede
configurar el tamano del cuadro mágico. Entiendase por tamano la cantidad
de filas o de columnas.

-----------------------------------------
